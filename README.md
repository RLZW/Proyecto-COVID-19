### Proyecto-COVID-19-MVC-SQL-Express
Proyecto desarrollado para la clase de desarrollo web con una base de datos montada en MySQL, backend en MVC con Express.js

### Correr el proyecto
`npm init`
`npm install express`
`npm install body-parser`
`npm install mysql2`
`node index.js`
